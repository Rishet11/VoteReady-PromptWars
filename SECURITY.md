# Security Policy

## Supported Versions

Currently, only the `main` branch (latest release) is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within VoteReady, please do not disclose it publicly. Instead, please report it via email to the project maintainers. We take all security vulnerabilities seriously and will work to address them immediately.

Upon receiving a vulnerability report, we will:
1. Acknowledge receipt within 48 hours.
2. Investigate the issue and determine its scope and impact.
3. If confirmed, develop a patch and release it as quickly as possible.
4. Publish a security advisory detailing the vulnerability and the fix.
